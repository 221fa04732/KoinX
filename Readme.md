# Welcome to KoinX

Deployed Link: https://koin-x-two-khaki.vercel.app/

---

## Local Setup

Download the repo: https://github.com/221fa04732/KoinX.git

### Backend:
- cd server  
- npm install  
- npm start  

### Frontend:
- cd client  
- npm install  
- change the `backend_url` to local development in `src/config.js`  
- npm run dev  

---

## Tech Stack Used

- ReactJS  
- Node.js  
- Recoil  
- Tailwind CSS  
- TypeScript
