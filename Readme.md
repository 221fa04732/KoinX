## Local Setup

Clone the repo: https://github.com/221fa04732/KoinX.git

### Backend:
- cd server  
- npm install  
- npm start  

### Frontend:
- cd client  
- npm install  
- change the `backend_url` to local development in `src/config.ts`  
- npm run dev
